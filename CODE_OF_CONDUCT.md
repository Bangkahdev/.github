# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and respectful environment for everyone.

## Expected Behavior

- Be respectful.
- Be constructive.
- Welcome newcomers.
- Focus on technical discussions.
- Accept feedback professionally.

## Unacceptable Behavior

- Harassment
- Hate speech
- Personal attacks
- Trolling
- Spam

## Enforcement

Project maintainers may edit, remove, or reject content that violates this Code of Conduct.

Thank you for helping build a healthy open-source community.
